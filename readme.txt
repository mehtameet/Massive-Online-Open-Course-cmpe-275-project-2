1. Go to Django server url (host:port/admin/) (login with username-meet and password-meet) inside mooc folder.

2. Select Classroom/sites

3. Add your service by entering groups’ name i.e. your MOOC’s name and IP address. (for our example its bottle running on localhost:8081)

4. Click it as a default URL and save it (make sure other default_url to unckeck else you run into errors).

5. Likewise, run the file which will execute the Bottle micro web framework which in turn will connect to the MongoDB database (inside mooc-ws).

6. go to host:port/home and play thorugh GUI.
